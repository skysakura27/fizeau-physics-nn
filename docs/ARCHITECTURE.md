# Fizeau Physics-Informed Neural Network Architecture

## 概述

本项目突破传统干涉解相算法与黑盒深度学习的瓶颈，实现 Fizeau 干涉仪 2nm 级别的超高精度、强鲁棒性波前测量。采用"由表及里"的物理闭环技术路径：

### 四阶段技术路径

#### 第一阶段：物理信号的获取与解构（问题定义）
- **核心任务**：分析 Fizeau 干涉图中的信号与核心噪声源
- **物理原理**：信号生成遵循 Airy 函数形态 $I = \frac{I_{max}}{1 + F \sin^2(\phi/2)}$
- **噪声机理**：
  - 散斑噪声 (Speckle)：高频随机干涉特征
  - 寄生条纹 (牛眼环)：中频空间结构

#### 第二阶段：频域解耦预处理（The Gatekeeper）
- **核心任务**：2D-DWT 通道分离，实现信号与散斑的解耦
- **物理动作**：抑制高频对角分量（HH），提取低频和中频分量（LL, LH, HL）
- **意义**：避免网络在拟合高频噪声上浪费算力

#### 第三阶段：物理嵌入式核心引擎（The Unrolled Engine）
- **核心任务**：Airy-PIN 迭代层（物理纠偏 + 残差清洗）
- **物理动作**：
  - 动作 A：Airy 函数梯度算子 $\nabla_{\phi} \mathcal{L}_{Airy}$
  - 动作 B：CNN 近端算子吸收非物理残差
- **意义**：实现"架构即物理"，杜绝泛化灾难

#### 第四阶段：拓扑硬约束输出（The Zernike Supervisor）
- **核心任务**：36 阶 Zernike 多项式系数预测
- **物理动作**：硬约束输出必须由标准物理面型线性叠加
- **意义**：锁定波前的纳米级"宏观骨架"

## 组件详解

### 1. DWT 预处理层（频域解耦）

**功能**: 对输入干涉图进行多尺度小波分解，实现信号与噪声的频域分离。

**实现**:
- 使用 Haar 小波基
- 2级分解 (低频 + 3个方向的高频)
- 输出: 近似系数 + 细节系数

**物理意义**:
- 抑制高频对角分量（HH）- 散斑噪声的主要藏身处
- 保留低频和中频分量（LL, LH, HL）- 包含有用信号
- 提前完成频域特征解耦，降低后续网络复杂度

**优势**:
- 多分辨率分析
- 保持相位信息
- 噪声鲁棒性

### 2. Airy-PIN 迭代层（物理嵌入式核心引擎）

**结构**: N 个串联的 Airy-PIN 层 (N=5-10)

**每个层执行两个物理动作**:
- **动作 A（物理纠偏）**: 硬编码 Airy 函数梯度算子 $\nabla_{\phi} \mathcal{L}_{Airy}$。根据多光束干涉的物理法则，计算当前相位猜测值带来的强度残差，并强制相位向符合物理光强分布的方向更新
- **动作 B（残差清洗）**: 利用轻量级 CNN 充当近端算子 (Proximal Operator)。由于物理公式无法描述系统中的随机热噪声和未知寄生环，CNN 负责吸收并剔除这些脱离 Airy 模型的"非物理残差"

**展开公式**:
```
φ_{k+1} = φ_k + ∇_{φ}ℒ_{Airy}(φ_k, I_meas) + CNN_{prox}(φ_k - φ_{k-1})
```

其中：
- $∇_{φ}ℒ_{Airy}$ 为 Airy 物理梯度
- $CNN_{prox}$ 为近端算子网络

**意义**: 实现了"架构即物理"，保证相位解调的每一步都在物理梯度的指引下进行，杜绝了纯数据驱动带来的泛化灾难。

### 3. 残差去噪网络

**架构**: U-Net 风格的编解码器

**特点**:
- 多尺度特征提取
- 跳跃连接保持空间信息
- 残差学习噪声模式

**集成方式**:
- 并行于主网络运行
- 在每个展开块后进行去噪
- 使用注意力机制选择性去噪

### 4. Zernike 监督器（拓扑硬约束输出）

**功能**: 网络末端放弃逐像素（Pixel-wise）相位输出，改为预测 36 阶 Zernike 多项式系数，最后通过矩阵乘法还原三维相位。

**物理原理 (像差正交基底投影)**: Zernike 多项式是在单位圆上正交的连续多项式集，是光学界描述波前像差（如离焦、像散、彗差）的"官方语言"和物理基底。

**物理动作**: 这是一个硬约束 (Hard Constraint)。强制网络预测的相位必须是由这 36 种标准物理面型线性叠加而成。

**优势**:
- 物理可解释性
- 拓扑结构约束
- 纳米级精度保证

**重建公式**:
```
φ(x,y) = ∑_{j=1}^{36} c_j × Z_j(ρ,θ)
```

其中：
- $c_j$ 为第 j 阶 Zernike 系数
- $Z_j(ρ,θ)$ 为第 j 阶 Zernike 多项式
- $(ρ,θ)$ 为单位圆坐标

**意义**: 从数学拓扑结构上彻底封死了网络输出断裂、突变或非物理高频抖动的可能性，锁定波前的纳米级"宏观骨架"。

## 训练策略

### 数据准备

**合成数据生成**:
- 随机 Zernike 系数组合
- 添加现实噪声模型
- 多种信噪比范围

**数据增强**:
- 随机旋转和翻转
- 尺度变换
- 噪声强度变化

### 损失函数

**多任务损失**:
```
L_{total} = w_1 * L_{data} + w_2 * L_{physics} + w_3 * L_{zernike} + w_4 * L_{denoise}
```

- **数据保真度损失**: MSE(预测干涉图, 真实干涉图)
- **物理损失**: 基于衍射模型的正则化
- **Zernike 损失**: 模态分解约束
- **去噪损失**: 噪声抑制质量

### 优化算法

**AdamW 优化器**:
- β1=0.9, β2=0.999
- 权重衰减: 1e-4
- 学习率调度: Cosine Annealing

**训练流程**:
1. 预热阶段 (10% 训练): 低学习率
2. 主训练阶段: 余弦学习率衰减
3. 微调阶段: 进一步降低学习率

## 推理流程

### 前向传播

1. **预处理**: DWT 分解输入干涉图
2. **初始化**: 使用传统方法粗略估计相位
3. **展开迭代**: N 次物理-神经联合更新
4. **去噪**: 应用残差去噪网络
5. **后处理**: Zernike 投影和相位展开

### 计算复杂度

- **空间复杂度**: O(N×C×H×W) , N=展开层数, C=通道数
- **时间复杂度**: O(N×C×H×W×K) , K=卷积核大小

### 内存优化

- **梯度累积**: 减少批次大小
- **混合精度**: FP16 训练
- **模型并行**: 多 GPU 分布式训练

## 性能评估

### 定量指标

- **PSNR**: 峰值信噪比 (目标 >30dB)
- **SSIM**: 结构相似性指数 (目标 >0.9)
- **RMSE**: 均方根误差 (目标 <0.05 rad)

### 定性评估

- **相位连续性**: 去除 2π 跳跃
- **表面精度**: PV 和 RMS 误差
- **噪声抑制**: 频域分析

### 鲁棒性测试

- **信噪比范围**: 10dB - 40dB
- **相位范围**: 0 - 10π rad
- **分辨率**: 256×256 - 1024×1024
