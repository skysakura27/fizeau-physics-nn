# 物理模型详解

## Fizeau 干涉仪原理

### 基本原理

Fizeau 干涉仪通过测量测试表面与参考表面之间的光程差来重建表面形貌：

```
光程差 (OPD) = 2 × 表面高度差 × 折射率
相位差 φ = (2π / λ) × OPD
```

其中：
- λ: 光波长 (通常 632.8nm)
- OPD: 光学路径差

### 干涉图形成

干涉图强度分布：

```
I(x,y) = I0 × [1 + γ(x,y) × cos(φ(x,y) + φ0)]
```

其中：
- I0: 背景强度
- γ(x,y): 干涉条纹对比度
- φ(x,y): 相位分布
- φ0: 参考相位

## Airy 衍射理论

### 点扩散函数 (PSF)

光学系统的 PSF 由 Airy 盘描述：

```
PSF(ρ) = [2 × J1(π × ρ) / (π × ρ)]²
```

其中：
- J1: 一阶贝塞尔函数
- ρ = r / (0.61 × λ / NA): 归一化径向坐标
- NA: 数值孔径

### 光学传递函数 (OTF)

频域响应：

```
OTF(f_x, f_y) = F{PSF(x,y)}
```

截止频率：
```
f_c = NA / λ
```

### 相位传递

在相干照明下，相位传递函数：

```
H(f_x, f_y) = rect(f_x² + f_y² ≤ f_c²)
```

## 相位重建算法

### 传统方法

#### 傅里叶变换法

1. 对干涉图进行 2D FFT
2. 选择一个谱峰
3. 相位提取：φ = arg(F⁻¹{谱})

#### 逐点相位展开

使用质量引导算法：
1. 计算质量图：Q = |I(x,y) - I0|
2. 沿最高质量路径展开相位
3. 最小二乘积分填充剩余区域

### 物理信息方法

#### 数据保真度

```
L_data = ||I_meas - I_pred(φ)||²
```

其中 I_pred 通过正向模型计算。

#### 相位平滑性

```
L_smooth = ∑ ||∇φ||²
```

#### 物理正则化

基于 Zernike 多项式的正则化：

```
L_zernike = ||φ - Z^T c||² + λ||c||₁
```

## Zernike 多项式

### 数学基础

Zernike 多项式是正交基函数：

```
Z_n^m(ρ,θ) = R_n^m(ρ) ×
    √(2(n+1)) ×
    { cos(mθ)    if m ≥ 0
    { sin(|m|θ)  if m < 0
```

径向多项式 R_n^m(ρ)：

```
R_n^m(ρ) = ∑_{k=0}^{(n-m)/2} (-1)^k ×
    [(n-k)! / (k! × ((n+m)/2 - k)! × ((n-m)/2 - k)!)] × ρ^{n-2k}
```

### 标准 Zernike 模态

| j | n | m | 名称 | 描述 |
|---|----|----|------|------|
| 0 | 0 | 0 | Piston | 常数项 |
| 1 | 1 | 1 | Tilt X | X 倾斜 |
| 2 | 1 | -1 | Tilt Y | Y 倾斜 |
| 3 | 2 | 0 | Defocus | 离焦 |
| 4 | 2 | 2 | Astigmatism 0° | 散光 0° |
| 5 | 2 | -2 | Astigmatism 45° | 散光 45° |
| 6 | 3 | 1 | Coma X | 彗差 X |
| 7 | 3 | -1 | Coma Y | 彗差 Y |
| 8 | 3 | 3 | Trefoil X | 三叶草 X |
| 9 | 3 | -3 | Trefoil Y | 三叶草 Y |
| 10 | 4 | 0 | Spherical | 球差 |

### 投影计算

相位到 Zernike 系数的投影：

```
c_j = <φ, Z_j> = (1/π) ∫∫ φ(ρ,θ) Z_j(ρ,θ) ρ dρ dθ
```

重构相位：

```
φ(ρ,θ) = ∑ c_j Z_j(ρ,θ)
```

## 噪声模型

### 系统噪声

#### 读出噪声

```
σ_readout = √(N_pixels × gain × (dark_current × t + readout_noise²))
```

#### 散粒噪声

```
σ_shot = √(signal_level)
```

#### 固定模式噪声 (FPN)

```
FPN(x,y) = offset(x,y) + gain(x,y) × signal(x,y)
```

### 相位噪声

#### 相位测量精度

```
σ_φ = σ_I / (I × dI/dφ) = σ_I / (I × √(γ) × sin(φ))
```

#### 相位展开误差

2π 跳跃检测误差导致的积累误差：

```
σ_unwrap = √(N_branches) × σ_φ
```

## 校正算法

### 像差校正

#### 低阶像差移除

```
φ_corrected = φ_measured - ∑_{j=1}^{10} c_j Z_j
```

#### 高阶像差保留

保留 j > 10 的高阶项用于表面形貌重建。

### 非线性效应校正

#### 检测器非线性

```
I_corrected = I_measured^(1/γ)
```

其中 γ ≈ 2.2 为 gamma 值。

#### 强度饱和校正

```
I_corrected = I_measured / (1 + I_measured/I_sat)
```

### 环境效应校正

#### 温度漂移

```
φ_corrected = φ_measured - α × ΔT × OPD
```

#### 机械振动

使用相位锁定或运动补偿算法。

## 数值实现

### 离散 Zernike 变换

#### 采样点生成

```
ρ_ij = √(x_i² + y_j²) / R
θ_ij = atan2(y_j, x_i)
```

#### 数值积分

使用高斯积分：

```
c_j = ∑ w_k Z_j(ρ_k, θ_k) φ_k
```

### FFT 加速

#### 快速相位重建

```
φ = arg(F⁻¹{shift(F{I × exp(iφ_ref)})} )
```

#### OTF 应用

```
φ_filtered = F⁻¹{ F{φ} × OTF }
```

## 性能分析

### 理论精度极限

#### 衍射极限

```
σ_min = λ / (2π × √(SNR))
```

#### 采样极限

```
σ_sample = λ / (2 × NA × pixel_size)
```

### 实际性能因子

#### 对比度损失

```
γ_eff = γ × (1 - σ_readout²/signal²)
```

#### 空间频率响应

```
MTF(f) = |OTF(f)| × γ_eff(f)
```
